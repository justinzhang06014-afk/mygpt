# Web Search Skill

本 Skill 提供联网搜索能力。

## 使用場景
- 需要查詢最新資訊時
- 需要搜索特定主題的網頁內容
- 需要驗證某個資訊的準確性

## 搜索方法

### 1. 使用 Web 搜索功能
當需要联网搜索時，直接使用以下工具：
- `web_search(query)` - 搜索網頁內容
- `web_fetch(url)` - 抓取指定網頁內容

### 2. 搜索技巧
- 使用精確的關鍵詞獲得更好結果
- 對於複雜查詢，拆分為多個簡單搜索
- 交叉驗證重要資訊來源

## 注意事項
- 搜索結果可能包含不準確資訊，需交叉驗證
- 尊重網站的 robots.txt 和版權規定
- 避免頻繁發送搜索請求

## 常見問題
Q: 搜索結果不完整？
A: 嘗試調整關鍵詞或使用不同的搜索詞組合。

Q: 無法訪問某些網站？
A: 可能是網路限制或網站反爬蟲機制，請稍後重試或換用其他來源。
